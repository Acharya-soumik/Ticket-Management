# Ticket-Management
## Features implemented:
1. Add movies
2. Select seats
3. Cancel Bookings
4. Filter by genre and language.
5. Block already booked tickets from booking again.

## Things Used to complete the task:
1. HTML
2. CSS
3. JavaScript
4. Local Storage
5. Bootstrap
